## Installation

Install all required libraries

```bash
  pip install -r requirements.txt
```
    
## Tech Stack

**Language:** Python

**AI Framework:** Pytorch

**Algorithms:** YOLOv5


## Features

- Upload multiple images
- Detect types of text in the images and crop out the detected parts
- Read the text using EasyOCR
